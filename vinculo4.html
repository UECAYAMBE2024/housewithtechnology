Instrucciones de Uso

Conexión del Módulo Bluetooth HC-05

Para conectar el módulo Bluetooth HC-05 al Arduino, siga estos pasos:

  1. Inserte el módulo Bluetooth en el protoboard.
  2. Conecte los pines RX y TX del módulo Bluetooth a los pines TX y RX del Arduino, respectivamente.
  3. Conecte el pin VCC del módulo Bluetooth a 5V y el pin GND a GND del Arduino.
  4. ¡Listo! El módulo Bluetooth está ahora conectado al Arduino y listo para ser utilizado.

Comandos para Funciones

A continuación se presentan los comandos para controlar las funciones del proyecto:

  • Encender LED: Enviar el número del LED (1-6) para encenderlo.
  • Apagar LED: Enviar la letra correspondiente al LED (a-f) para apagarlo.
  • Encender Todos los LEDs: Enviar el carácter 'Z' para encender todos los LEDs.
  • Apagar Todos los LEDs: Enviar el carácter '0' para apagar todos los LEDs.
